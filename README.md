# BACnetClientExamplePython2.7
## Installing Requirements
### Windows
```cmd
pip install -r requirements.txt
```

### Linux

```bash
pip2 install -r requirements.txt
```
## Running the client
### Windows
```ps
python BACnetClientExample.py {IPAddress}
```
### Linux
```bash
python3 BACnetClientExample.py {IPAddress}
```
